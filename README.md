\# Sales Analysis Project



This project processes two CSV files using \*\*Pandas\*\* to extract key insights and answer core business questions.



\## 🚀 Features

\* Data Integration & Cleaning: Automated loading and preprocessing of raw sales datasets.

\* Business Intelligence Metrics:

&nbsp;   \* Top-Selling Product Analysis: Identifies the highest-selling item and its total revenue contribution

&nbsp;   \* Comparative Performance: Calculates sales variance between December and the annual peak-performance month.

&nbsp;   \* Underperformance Tracking: pinpoints the product with the lowest sales volume and its respective month.

&nbsp;   \* Payday Impact Analysis: Computes bi-weekly sales averages to evaluate the correlation between paydays and consumer behavior. 


\## 🛠️ Technologies

\* Python 3.x

\* Pandas (Data manipulation)

\* Jupyter Notebooks
